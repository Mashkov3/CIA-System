# CIA Secure System

واجهة ويب متكاملة لقطاع الاستخبارات (خيالي)، نسخة تجريبية مع تخزين محلي، تحرير مباشر للجداول، تصدير تقارير، وتعديل البيانات بشكل ديناميكي.  
**ملاحظة:** هذا المشروع ولا يتصل بأي قاعدة بيانات خارجية أو أنظمة حقيقية.

---

## الميزات الرئيسية

- **تسجيل الدخول الآمن**  
  - تهيئة Admin لأول مرة وحفظ الكود داخل المتصفح.
  - الدخول مباشرة باستخدام اسم الموقع وكود المرور.

- **لوحة تحكم HUD سينمائية**  
  - عرض إحصائيات العمليات، الرتب، المطلوبين بشكل ديناميكي مع تأثير العد التنازلي للأرقام.
  - أشرطة تقدم ومستوى تهديد عالمي.

- **الجداول المتقدمة**  
  - الحالات العسكرية (خيالية)، الرتب، المطلوبون.  
  - **تعديل مباشر** على الخلايا.  
  - **حذف أي صف** بسهولة.  
  - **إدراج صورة لكل صف**.  
  - **بحث وفلترة مباشرة**.  
  - التواريخ الافتراضية تُملأ تلقائيًا عند إضافة صف جديد ويمكن تعديلها يدويًا.

- **تصدير التقارير**  
  - PDF و Word لجميع البيانات.

- **تخزين محلي بالكامل**  
  - كل البيانات تُخزن داخل المتصفح (`localStorage`).  
  - لا حاجة لخادم خارجي.

- **واجهة مظلمة حديثة**  
  - تصميم مستوحى من واجهات الاستخبارات السينمائية.  
  - خطوط تقنية وألوان HUD.

---

## كيفية التشغيل

1. احفظ ملفات المشروع في مجلد واحد على جهازك.  
2. افتح ملف `index.html` في أي متصفح حديث (Chrome، Firefox، Edge).  
3. أول مرة، اضغط **تهيئة Admin لأول مرة** مع إدخال كود المرور الذي تريده.  
4. أدخل اسم الموقع وكود المرور واضغط **دخول**.  
5. الآن تستطيع استخدام كل الميزات:

   - إضافة/تعديل/حذف حالات، رتب، المطلوبون.  
   - إدراج صور لكل صف.  
   - البحث والفلترة.  
   - تصدير PDF أو Word.

---

## ملاحظات مهمة

- هذه النسخة **تجريبية بدون تشفير SHA-256**، لذلك الكود يُخزن نصيًا داخل المتصفح.  
- لتفعيل أمان أقوى (SHA-256)، يجب تشغيل الصفحة عبر **سيرفر محلي أو HTTPS**.  
- جميع البيانات محفوظة في `localStorage` الخاص بالمتصفح، ولن تنتقل لأجهزة أخرى.  

---

## التقنيات المستخدمة

- HTML / CSS / JavaScript  
- LocalStorage لتخزين البيانات  
- [jsPDF](https://cdnjs.com/libraries/jspdf) لتصدير PDF  
- [docx.js](https://cdnjs.com/libraries/docx) لتصدير Word  

---

## تعليمات إضافية

- استخدم **Live Server** في VS Code للحصول على تجربة أفضل عند تفعيل SHA-256 لاحقًا.  
- كل تعديل يتم في الجداول يتم حفظه تلقائيًا في المتصفح.  
- يمكنك إضافة أي أعمدة إضافية أو تعديل تصميم HUD حسب الحاجة.
